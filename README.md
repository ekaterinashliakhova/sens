# Sensorium — Интерактивный чат-бот для продажи билетов

## Описание проекта

Sensorium — это веб-приложение в формате чат-бота для продажи билетов на мероприятия в темноте. Приложение предоставляет интерактивный интерфейс для выбора сценариев, дат, времени и количества билетов.

## Функциональность

### Основные разделы:
- **Приобрести билет** — покупка билетов на основные сценарии
- **Свидания** — романтические сценарии в темноте
- **Мероприятия** — специальные события
- **FAQ** — часто задаваемые вопросы
- **Экскурсия** — информация об экскурсиях
- **Отзывы** — отзывы посетителей
- **Об исследованиях темноты** — научные факты о влиянии темноты

### Особенности:
- Интерактивный чат-интерфейс
- Выбор даты и времени посещения
- Подбор количества билетов (взрослые/детские)
- Слайдеры с отзывами и исследованиями
- Адаптивный дизайн

## Установка и запуск

1. Откройте файл `index.html` в любом современном браузере
2. Для локальной разработки рекомендуется использовать локальный сервер:
   ```bash
   # Используя Python 3
   python -m http.server 8000
   
   # Используя Node.js (http-server)
   npx http-server
   ```
3. Перейдите по адресу `http://localhost:8000`

## Архитектура приложения

### State Management
Приложение использует простое состояние для управления навигацией:
- `chatOpen` — статус открытия чата
- `history` — стек истории для навигации назад
- `currentScreen` — текущий экран
- `orderData` — данные заказа
- `formData` — данные форм

### Компоненты:
1. **Аватар** — визуальный элемент чат-бота
2. **Чат-сообщения** — область отображения сообщений
3. **Кнопки действий** — интерактивные элементы выбора
4. **Слайдеры** — карусели для отзывов и исследований
5. **Дата-пикер** — выбор даты и времени

## Разработка

### Добавление нового раздела:
1. Создайте функцию `showSectionName()` в `script.js`
2. Добавьте обработку сообщений и кнопок
3. При необходимости создайте слайдер с данными
4. Добавьте кнопку меню в главный экран

### Добавление отзывов:
```javascript
const reviews = [
  {
    place: 'Яндекс Карты',
    name: 'Имя Фамилия',
    text: 'Текст отзыва',
    link: '#ссылка'
  }
];
showSlider(reviews.map(r => ({...r, isReview: true})));
```

### Стилизация:
Все стили находятся в `style.css`. Основные классы:
- `.avatar` — аватар бота
- `.chat-messages` — контейнер сообщений
- `.action-buttons` — кнопки действий
- `.slider-container` — контейнер слайдера
- `.review-card` — карточка отзыва
